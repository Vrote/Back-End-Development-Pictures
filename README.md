# Back-End Development - Pictures Microservice

This repository contains the Pictures microservice, responsible for managing image metadata and storage for the Band Fan Club application.

## Environment Setup
- **Created from template:** Yes
- **Successful execution of bin/setup.sh:** Yes
- **Python Version:** 3.9.x
- **Virtual Environment Name:** backend-pictures-venv

## Technologies Used
- **Python/Flask:** Backend framework
- **MongoDB:** Database for picture metadata
- **Pytest:** Unit and integration testing
